線形代数：核と像、および完全列

1. 核 \(\ker(f)\) と像 \(\mathrm{im}(f)\) の役割

核 \(\ker(f)\)

線形写像 \(f: V \to W\) の核 \(\ker(f)\) は、\(W\) においてゼロに写される \(V\) のベクトルの集合です。正式には、 \[ \ker(f) = \{v \in V \mid f(v) = 0\}. \] つまり、\(\ker(f)\) は \(f\) によって「消える」ベクトルからなる部分空間です。

対称性の観点: \(\ker(f)\) は、写像 \(f\) が無視または破壊する対称性（または情報）を表します。

像 \(\mathrm{im}(f)\)

像 \(\mathrm{im}(f)\) は、\(f\) の全ての出力の集合を表します： \[ \mathrm{im}(f) = \{f(v) \mid v \in V\}. \] これは、写像 \(f\) によって生成される \(W\) の部分空間です。

対称性の観点: \(\mathrm{im}(f)\) は、写像 \(f\) によって保存および伝播される対称性を表します。

2. 完全列

完全列 は、次のようなベクトル空間と線形写像の列です： \[ 0 \longrightarrow A \xrightarrow{f} B \xrightarrow{g} C \longrightarrow 0 \] ここで、各写像の像が次の写像の核に等しいことを意味します。具体的には： \[ \mathrm{im}(f) = \ker(g). \]

対称性の観点: 完全列は、対称性が空間をどのように移動するかを追跡します。どこで対称性が保存され、変換され、または「失われる」（ゼロに写される）かを示します。

3. 対称性に関する直感的な理解

  • 核 \(\ker(f)\): 写像 \(f\) によって「消える」対称性の部分空間。
  • 像 \(\mathrm{im}(f)\): 写像 \(f\) によって「保存される」またはターゲット空間に表現される対称性。
  • 完全列: 対称性が空間をどのように移動し、どこで情報が失われたり変換されたりするかを記述します。

4. 具体例

線形写像 \(f: \mathbb{R}^3 \to \mathbb{R}^2\) を次のように定義します：

\[ f(x, y, z) = (x + y, y + z). \]

核 \(\ker(f)\) は次の条件を満たすベクトル \((x, y, z)\) の集合です： \[ x + y = 0 \quad \text{および} \quad y + z = 0. \] これを解くと、\(\ker(f) = \{(t, -t, -t) \mid t \in \mathbb{R}\}\) という1次元部分空間になります。

像 \(\mathrm{im}(f)\) は、\((1, 1)\) と \((0, 1)\) によって張られる \(\mathbb{R}^2\) の部分空間です。この場合、\(\mathbb{R}^2\) 全体が像となります。

5. 結論

線形写像の核と像、そして完全列の概念は、空間間で対称性や構造がどのように変換されるかを理解するための強力なツールです。核は「失われる」部分を示し、像は「保存される」部分を示し、完全列はこれらの概念を結びつけます。

[解説・用語]

線形代数で習う**核（ker(f))と像（im(f))の完全列（exact sequence）**については、群論での準同型定理と似た概念を含みますが、より具体的に「線形写像」に関する構造を捉えるためのツールです。これを対称性の観点から理解する方法について説明します。
